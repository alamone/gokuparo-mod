# gokuparo-mod
Gokujou Parodius mod

Skip to any desired stage (Special stage by default) and set to max rank.
I might expand this mod in the future to have more features.

Instructions:
- Copy "321jad02.21b" and "321jad04.27b" into the "gokuparo" folder.
- Run "patch.bat".
- Output will be in the "gokuparo-modded" folder.
